 
SettleMate is a Java-based Expense Sharing and Settlement System designed to help groups manage shared expenses efficiently. The system allows users to create groups, add members, record expenses, and automatically calculate settlements between members. It aims to reduce confusion and manual calculation errors that occur when managing shared expenses during trips, hostel life, or group activities. 
The application is built using object-oriented programming principles in Java. The system uses a modular architecture consisting of models, services, strategies, and utility components. Users can split expenses using different methods such as equal split, exact split, and percentage split. These split types are implemented using the Strategy Design Pattern, allowing  correct splitting when the share/contribution in activity is not same by all members.   
The project also includes features such as budget tracking, spending analytics, and data persistence using Java serialization. The system stores data using database so that user and expense information remains saved.
